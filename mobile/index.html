Java后台，即Java EE。使用Java EE能够开发一个完整的Web应用，但是由于前后端的分离， 所以Java EE现在一般用于后台的开发。如果你足够的优秀，那么你能够成为一个全栈工程师。 Java EE拥有强大的跨平台的能力，它能够使你在windows上开发应用，部署在Linux的服务器上。 Java EE拥有小巧玲珑的Servlet/JSP，使用它能够迅速的开发一个小型的应用。Java EE不仅于Java EE， 它拥有众多的框架，如贯穿整个应用的Spring，简化数据库的Hibernate、Mybaits等，关于安全的Spring Security、 Shiro等。这些框架能够帮助你开发一个安全可靠的、分层明确的、便用维护的应用。
